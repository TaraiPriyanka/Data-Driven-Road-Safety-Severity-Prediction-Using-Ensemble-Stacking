Accident Severity Prediction System

A Machine Learning-Based Web Application

About the Project

Road traffic accidents are a major cause of injury and death worldwide. This system predicts the severity of a road accident based on real-time inputs like weather, road type, time, and traffic conditions using machine learning. The prediction helps in:

  • Early emergency response dispatching
  • Prioritizing high-risk areas
  • Making data-driven decisions for traffic safety

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Flask (Python)
  • Machine Learning: Voting Classifier combining Logistic Regression, Decision Tree, and Random Forest
  • Data Preprocessing: Encoding, Feature Scaling, Null Handling
  • Prediction Output: Classified as Fatal, Serious, or Slight
Prediction Classes
Severity Class Meaning Impact
Fatal Accident leads to death High-priority emergency required
Serious Severe injury but non-fatal Ambulance & hospital involvement
Slight Minor injuries or vehicle damage General road assistance
